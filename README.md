# IRCTC Next Frontend

Ultra-modern IRCTC-like frontend built with React 18 + Vite + Tailwind CSS + Zustand.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Highlights

- Landing, search, booking, profile, admin dashboard pages
- Razorpay/Stripe-ready payment modal scaffold
- Notification snackbar + loading skeletons
- Dark/light theme toggle
- PWA basics (`manifest.json`, `service-worker.js`)
- Docker + Kubernetes manifests

## Environment variables

Use `.env.example`.
